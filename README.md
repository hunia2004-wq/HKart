# HKart - Mobile Commerce App

A full-stack mobile shopping application built with React Native Expo and Supabase.

## Scenario A — Shop App

## Tech Stack
- React Native with Expo SDK 54
- Supabase (Auth, Database, Storage)
- React Navigation v6
- Zustand (State Management)

## Setup Instructions

1. Clone the repository
2. Install dependencies
3. Create a `.env` file based on `.env.example`
4. Run the app
## Features
- User Authentication
- Product Listing with category filters
- Product Image Upload
- Barcode Scanner
- Shopping Cart
- Push Notifications
- Store Locator
- Offline Browsing
- Receipt PDF Export
- Secure Checkout
- Network Status Banner
- Battery-Aware Sync
- Order History
- Wishlist
- Haptic Feedback
- Deep Linking
